# UNP_SQL_PROJECT_E_COMMERCE_STORE_DATASET_ANALYSIS

DESCRIPTION :This project is a MySQL-based E-Commerce Database Management System that manages customers, products, stock, orders, payments, and shipping details. It implements relational database concepts using primary and foreign keys to maintain data integrity. The system supports CRUD operations and ALTER commands, demonstrating real-world database design and efficient data handling.
